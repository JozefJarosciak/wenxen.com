Client-Side Diagram Alternatives (Since D2 needs server compilation)

Option 1: Viz.js (Graphviz in Browser)

Graphviz compiled to WebAssembly, runs entirely client-side

  • ‚úÖ No server needed
  • ‚úÖ Industry standard DOT language
  • ‚úÖ Excellent layout algorithms
  • ‚úÖ 300KB library size
  • ‚ùå Requires loading WASM
  • ‚ùå DOT syntax learning curve

Implementation: <script src="https://unpkg.com/viz.js">

Option 2: Cytoscape.js

Graph theory visualization library

  • ‚úÖ Pure JavaScript
  • ‚úÖ Interactive diagrams
  • ‚úÖ Many layout options
  • ‚úÖ Highly customizable
  • ‚ùå 400KB library
  • ‚ùå More complex API

Option 3: Nomnoml

Simple, hand-drawn style diagrams

  • ‚úÖ Only 60KB
  • ‚úÖ Simple syntax
  • ‚úÖ Unique visual style
  • ‚úÖ No dependencies
  • ‚ùå Limited diagram types
  • ‚ùå Less professional look

Implementation: <script src="https://unpkg.com/nomnoml">

Option 4: Flowchart.js

Specifically for flowcharts

  • ‚úÖ Only 30KB
  • ‚úÖ Simple text syntax
  • ‚úÖ Clean output
  • ‚ùå Only flowcharts
  • ‚ùå Limited customization

Option 5: Pure SVG (Current Best)

Hand-coded or generated SVG diagrams

  • ‚úÖ Zero dependencies
  • ‚úÖ Perfect control
  • ‚úÖ Smallest size
  • ‚úÖ CSS styleable
  • ‚úÖ Works everywhere
  • ‚ùå More manual work
  • ‚ùå No automatic layout

Option 6: ELK.js

Eclipse Layout Kernel in JavaScript

  • ‚úÖ Professional layouts
  • ‚úÖ Used by VS Code
  • ‚úÖ Highly configurable
  • ‚ùå 200KB size
  • ‚ùå Complex API

üéØ RECOMMENDATION: Viz.js or Pure SVG

For your use case:

  • If you want automatic layout: Viz.js (Graphviz in browser)
  • If you want zero dependencies: Pure SVG (what I started implementing)
  • If you want simple syntax: Nomnoml (60KB, unique style)

D2 is great but requires either:

  • Server-side compilation
  • Calling their playground API (requires internet, may have rate limits)
  • Running D2 CLI locally (not browser-based)

Quick Viz.js Example:

<script src="https://unpkg.com/viz.js"></script>
<script>
  const dot = `
    digraph G {
      rankdir=TD;
      node [shape=box, style="rounded,filled", fillcolor="#e3f2fd"];
      
      start [label="Start Scan"];
      check [label="Got Blocks?", shape=diamond];
      chunk [label="Chunk em up"];
      done [label="Victory Dance"];
      
      start -> check;
      check -> chunk [label="Yes"];
      check -> done [label="No"];
    }
  `;
  
  const viz = new Viz();
  viz.renderSVGElement(dot).then(svg => {
    document.getElementById('diagram').appendChild(svg);
  });
</script>
